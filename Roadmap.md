# Roadmap: Neural Fluidity Kernel (NFK)

- [ ] **Phase 1: Optimization** - Implement dynamic shape support for variable hidden dimensions.
- [ ] **Phase 2: Compatibility** - Add support for FP16 and BF16 precision (Essential for modern LLMs).
- [ ] **Phase 3: Integration** - Create a PyTorch `nn.Module` wrapper for easy "drop-in" replacement.
- [ ] **Phase 4: Benchmark Suite** - Comprehensive performance comparison against cublas and cutlass.
